# اسکنر و بررسی‌کننده SNI Spoof 🕵️‍♂️⚡

یک ابزار گرافیکی (GUI) قدرتمند، سریع و غیرهمزمان (Asynchronous) که با استفاده از پایتون و Tkinter توسعه یافته است. این برنامه برای اسکن ساب‌نت‌های IP (IP Subnets) بر روی هاست‌نیم‌های خاص SNI کاربرد دارد. این ابزار به محققان شبکه و توسعه‌دهندگان کمک می‌کند تا نقطه‌های پایانی (Endpoints) فعال TLS و «آی‌پی‌های تمیز (Clean IPs)» را با کمترین تأخیر (Ping) پیدا کرده و از سیستم‌های تأییدیه استاندارد هاست‌نیم عبور کنند.

## ✨ ویژگی‌ها

* **اسکن غیرهمزمان (Asynchronous):** بهره‌گیری از `asyncio` برای همزمانی بالا، که به شما اجازه می‌دهد هزاران IP را در چند ثانیه اسکن کنید.
* **رابط کاربری تاریک و مدرن (Dark UI):** طراحی اختصاصی و چشم‌نواز Tkinter که برای استفاده‌های طولانی‌مدت بهینه شده است.
* **نتایج در لحظه (Real-Time):** جدول وضعیت زنده که نتایج را بر اساس میزان تأخیر (سریع/متوسط/کند) با نشانگرهای بصری و رنگی دسته‌بندی می‌کند.
* **پارامترهای کاملاً قابل شخصی‌سازی:** امکان تنظیم آسان Timeout (زمان انتظار)، Concurrency (تعداد درخواست‌های همزمان) و Port (پورت پیش‌فرض `443`) به صورت مستقیم از داخل برنامه.
* **خروجی‌گیری خودکار:** ذخیره خودکار IPهای سالم و دارای TLS به همراه پینگ آن‌ها در فایل `working_tls_ips.txt`.
* **بدون نیاز به پیش‌نیازهای جانبی:** این ابزار کاملاً با کتابخانه‌های استاندارد پایتون نوشته شده است؛ **هیچ نیازی** به `pip install` ندارید!

## 🛠️ راهنمای استفاده

1. ریپازیتوری را کلون (Clone) کنید یا فایل اسکریپت را دانلود نمایید.
2. اسکریپت را از طریق ترمینال (Terminal) یا Command Prompt اجرا کنید:

    ```bash
    python checker.py
    ```

3. **کانفیگ اسکن خود را تنظیم کنید:**
    * **IP SUBNETS:** رنج آی‌پی‌های هدف خود را وارد کنید (مثال: `104.19.229.0/24`). می‌توانید چند ساب‌نت را در خطوط جداگانه (با Enter) وارد کنید.
    * **SNI HOSTNAMES:** نام‌های دامنه SNI مورد نظر را وارد کنید (مثال: `www.cloudflare.com`).
    * **Settings (تنظیمات):** مقادیر Timeout، Concurrency و پورت را بر اساس قدرت سیستم و اینترنت خود تنظیم کنید.
4. روی دکمه **▶ START SCAN** کلیک کنید و منتظر دریافت نتایج بمانید.
